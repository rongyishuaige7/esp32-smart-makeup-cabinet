# 安全、网络、数据与不适用场景

## 严格不适用场景

- 本项目不是化妆品质量、真假、保质期真实性、皮肤健康、紫外线防护、医疗、环境安全、门禁、电子锁、防盗、身份认证、访问控制、应急响应或无人值守系统。
- 不得依据温湿度、PIR、未校准 UV 模块输出、固定阈值、RFID UID、OLED、串口、HTTP JSON、App、CI 或构建结果作出化妆品可用性、健康、安全、人员在场、门锁、权限、告警送达或有人处理的结论。
- 不得将本项目用于市电、危险场所、贵重物保管、人员安全、消防/报警、门锁/防盗、远程控制、生产控制或任何需要可靠反馈、认证、授权、审计和人工处置的场景。

## 公开默认与本地 opt-in

- 本地 `firmware/src/config.local.h` 被 Git 忽略，仅可用于个人受监督、低压、隔离可信局域网台架实验。它不是安全边界，也不授予任何远程控制、门锁、认证或生产使用能力。
- `accepted:true` 只表示 HTTP handler 接受本次请求；`*Commanded*` 是软件命令变量；`localControlAndActuatorsEnabled` 只表示两个实验开关同时启用；`publicDefaultInert` 是它的反值，不能概括 RFID 维护、设备在线或实体输出；`physicalPositionVerified=false` 明确表示没有执行器或舵机物理位置反馈。

## 凭据、网络与数据

- `firmware/src/config.local.h` 只能保存在本机。绝不提交、截图、录屏、粘贴到 Issue、写入日志或以任何形式公开 Wi-Fi SSID、密码、私网 IP/MAC。
- HTTP 无 TLS、认证、会话、授权、设备身份、审计或速率限制。只有本地同时启用 `ENABLE_EXPERIMENTAL_ACTUATORS=1`、`ENABLE_EXPERIMENTAL_LOCAL_CONTROL=1`、提供非空 Wi-Fi 信息且成功入网后才可能启动；只可用于短期、隔离、可信、受监督教学局域网。禁止公网暴露、端口转发、公共 Wi-Fi、共享热点或不可信网络。
- App 保存的目标地址、本地物品名称/品牌/日期、网络日志、串口输出、实物照片与演示视频都可能泄露个人信息、位置或私网资料。请在公开前删除或脱敏这些材料及 EXIF/GPS。
- RFID UID 可复制，不能作为安全凭据。公开 HTTP 不返回 UID，固件不根据刷卡自动命令插销；本地物品记录、手工录入日期和 RFID 维护操作不构成可信商品数据库或安全认证。
- 一次“HTTP 读取成功”只表示当次 `/api/sensors` 与 `/api/status` 得到可解析 JSON；不表示设备在线、持续连接、传感器准确、数据实时、执行器动作或有人处理。地址变化或读取失败时 App 会清空旧报告。Android 普通 release manifest 不声明 `INTERNET` 或 cleartext 网络能力；debug/profile 为开发期本地 HTTP 演示保留网络与 cleartext 权限，不能被误当作发布版网络限制或真机联调的验证。

## 硬件边界

- ESP32 GPIO 与 ADC 输入不得超过 3.3 V；所有模块都须以实物数据确认额定电压、电平、电流、限流、上拉、驱动、保护、公共地与实际接线。
- 不要让 GPIO 直接驱动大电流负载。风扇、加湿器、WS2812B 和舵机必须使用合规低压供电、适当驱动与保护；舵机/机构有夹伤、卡滞和误动作风险。
- 未完成当前 commit 的实物复测前，不能将源码引脚、阈值、HTTP、App 状态或软件命令写成已验证接线、可靠网络、机械位置或安全功能。

## 报告问题

请勿在公开 Issue 中提交真实凭据、私网信息、位置、个人物品记录、照片元数据、真实传感器数据、网络抓包或串口日志。仓库没有承诺私密报告入口；请只提交最小化、已脱敏的复现材料。
